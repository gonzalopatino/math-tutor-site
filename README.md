# Math Tutor Marketing Website

## Overview

This repository contains the frontend application for a math tutor marketing website. The goal is to provide a clean, professional web presence that can be extended with content, a contact flow, and basic analytics.

## Architecture (High-Level)

- Frontend-only React application (static site) for the initial version.
- Future evolution may include an optional backend/API for contact forms, lead capture, analytics, and operational monitoring.

## Tech Stack

- React
- TypeScript
- Vite

## Local Development

To run the project locally:

```bash
npm install
npm run dev
```

The terminal will print a local URL, commonly http://localhost:5173, where the application can be accessed.

## Project Standards

This project is intended to follow professional software engineering standards:

- Linting and formatting will be enforced to maintain code quality and consistency
- Unit tests will be added and required as part of the development workflow
- Continuous Integration (CI) will validate linting, testing, and builds on every pull request
